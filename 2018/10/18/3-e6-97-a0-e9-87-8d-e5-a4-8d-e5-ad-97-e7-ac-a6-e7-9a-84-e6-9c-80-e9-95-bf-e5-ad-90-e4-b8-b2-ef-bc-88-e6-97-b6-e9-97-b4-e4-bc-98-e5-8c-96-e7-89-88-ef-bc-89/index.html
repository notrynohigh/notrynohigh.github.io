3.无重复字符的最长子串（时间优化版）

暴力解法：http://www.yourmcu.com/2018/09/28/longest-substring-without-repeating-characters/

在暴力计算后，由于耗时太长，转变思路：

1. 长度为0 或者 1直接返回长度即可

2. 假定无重复字符的字符串最长为 max_len 字符串总长为 s_len

3. max_len 取值范围是 2~max_len

4. 当max_len为2，找出是否有满足要求的子字符串，如果没有那么返回 max_len - 1 是最终结果

5. 如果找出了满足要求的子字符串，max_len加1后再进行查找

6. max_len加1不重头开始查找，而是在当前位置继续向后查找。即节省重复查找的时间。

代码：

int lengthOfLongestSubstring(char* s) {
int s_len = strlen(s);
int i = 0, j, max_len = 2, k, off = 1;

if(s\_len == 1 || s\_len == 0)
{
    return s_len;
}
for(;;)
{
    for(;i <= (s\_len - max\_len);i++)
    {
        for(j = off;j < max_len;j++)
        {
            for(k = 0;k < j;k++)
            {
                if(s\[i + j\] == s\[i + k\])
                {
                    break;
                }
            }
            if(k < j)
            {
                break;
            }
        }
        if(j >= max_len)
        {
            off = max_len;
            max_len++;
            break;
        }
        else
        {
            i += k;
            off = j - k;
        }

    }
    if(i > (s\_len - max\_len))
    {
        return (max_len - 1);
    }
}
return 0;

}

暴力求解方式耗时是500ms左右，上述代码耗时16ms
